# NullHooks
**CS:GO cheat I made based on the famous base, and many other cheats I found.**

#### Visuals - ESP
- Enable/disable team ESP
- Box ESP
- Name ESP
- Nade projectile ESP

- Spectator list
- Stats watermark (*username, time, fps, speed...*)
