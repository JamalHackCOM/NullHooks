# NullHooks
**CS:GO cheat I made based on the famous base, and many other cheats I found.**

Based on: [Click me](REFERENCES.md)

## Features
#### Visuals - ESP
- Enable/disable team ESP
- Player glow
- Box ESP
- Player skeleton
- Name ESP
- Health ESP
<!--
- Player info (flashed, weapons, etc.)
-->
- Line

#### Visuals - Others
- C4 glow
- Nade projectile ESP
- Grenade prediction
- No flash
- No scope
- Custom crosshair
- Recoil crosshair
<!-- 
- C4 timer
-->


#### Misc:
- BunnyHop
- Spectator list
- Stats watermark (*username, time, fps, speed...*)
